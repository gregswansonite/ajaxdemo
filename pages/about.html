About

Ajax Demo was built using jQuery for the front-end and PHP for the back-end. The layout and styling uses Bootstrap. All of the data is stored on a MySQL database. All of the code for this project is on GitHub.

This app allows a user to add contacts and then submit orders. Users are not allowed to enter an order unless there is an existing contact saved. Contacts and orders can be removed using the red buttons on each record.

All of the data entries and deletions are done using jQuery ajax requests. The back-end API consists of a few PHP files that take insertion, deletion and display requests. The API also returns all of the records in each table, in JSON format.
